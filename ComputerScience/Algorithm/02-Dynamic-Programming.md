# 동적 계획법이란

이전 단계의 계산 결과를 (다시) 이용해서 결과를 구하는 방법이다.
문제를 여러 개의 하위 문제로 나눈 뒤, 다른 하위 문제를 계산할 때 같은 연산이 필요할 경우 이미 연산해 저장해 둔 값을 이용한다.

점화식을 만들어서 문제를 해결한다고 생각할 수 있다.

<br><br>
